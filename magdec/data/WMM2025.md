# Coefficient files

Drop coefficient files into this directory and they will be auto-detected on
the next import.  The registry (`magdec/_registry.py`) scans for known
filenames in priority order; add a new entry there if the filename is not
already listed.

## Currently bundled

| File                | Model       | Epoch      | Max degree | Format | Source |
|---------------------|-------------|------------|------------|--------|--------|
| `WMM2010.COF`       | WMM-2010    | 2010–2015  | 12         | COF    | NOAA   |
| `WMM2015v2.COF`     | WMM-2015v2  | 2015–2020  | 12         | COF    | NOAA   |
| `WMM2020.COF`       | WMM-2020    | 2020–2025  | 12         | COF    | NOAA   |
| `WMMHR.COF`         | WMMHR-2025  | 2025–2030  | 133        | COF    | NOAA   |
| `igrf14coeffs.txt`  | IGRF-14     | 1900–2030  | 13         | SHC    | BGS/NOAA |

## Auto-selection order

Auto mode (`model='auto'`) picks the first available model for the date:

```
WMMHR2025  → 2025–2030  (high-resolution, n=133)
WMM2020    → 2020–2025
WMM2015v2  → 2015–2020
WMM2010    → 2010–2015
IGRF14     → 1900–2010  (pre-WMM coverage)
```

IGRF-14 also covers 2010–2030 and can be selected explicitly
(`model='IGRF14'`), but WMM is preferred for modern dates.

## Downloading newer files

### WMM (COF format)
All WMM coefficient files are public domain:
  https://www.ngdc.noaa.gov/geomag/WMM/soft.shtml

Standard WMM (n≤12): fast, suitable for navigation and batch processing.
WMMHR (n≤133): ~54× slower per point, captures crustal field structure;
  differences vs standard WMM are typically 0.05–0.3° depending on location.

### IGRF (SHC format)
IGRF-14 coefficient file from BGS / NOAA NCEI:
  https://www.ngdc.noaa.gov/IAGA/vmod/coeffs/igrf14coeffs.txt

The file is parsed by `magdec/_igrf.py` (BGS/NOAA SHC format).  Linear
interpolation is used between 5-year definitive epochs; the SV column
extrapolates to 2030.
